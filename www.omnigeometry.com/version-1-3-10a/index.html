Version 1.3.10a – Maintenance release

February 1, 2022

Version 1.3.10a is a maintenance release for fixing bugs left in in the previous release and to fix offline version license validation to work again. Yesterday our online licensing service made some changes that made the license validation and re-validation requests to fail, this version fixes that.

Online version will be updated soon to match.

For full list of changes in 1.3.10, see the Release Notes for 1.3.10. 

Release notes

Fix exporting vector PDF or SVG with transparent background.
Remove scene loading confirmation in offline version when scene modified, main window closed and loading a new scene from the recents menu or by double clicking on a scene file.
Fix saving to work when importing scene from text.
Fix point arc angle becoming NaN when loading a scene saved with an earlier version of OmniGeometry. 
Fix offline version license validation to work.
